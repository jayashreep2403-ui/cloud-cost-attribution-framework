**cloud-cost-attribution-framework
AWS-based cost attribution and tagging framework built with Terraform, CloudWatch, and Grafana.

## Problem

Cloud costs become difficult to track when AWS resources are not tagged consistently.

This project implements a standardized tagging framework to enable:

* Cost allocation by environment
* Cost allocation by application
* Cost allocation by owner
* FinOps governance
* Better cloud visibility

## Technologies

* AWS
* Terraform
* CloudWatch
* Grafana
* IAM

## Features

* Automated tagging standards
* Cost attribution strategy
* Infrastructure as Code
* Governance controls
* Monitoring dashboards
